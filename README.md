HORTON-GRID info:
--------------

HORTON-GRID is the integral grid implemented in Horton 2, but now it works with python 3.10+

Installation
------------

- git clone git@github.com:yingxingcheng/horton-grid.git
- cd `horton-gird`
- ./install.sh
